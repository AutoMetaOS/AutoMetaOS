# AutoMetaOS v7 (MacOS)

![AutoMetaOS](./amos.png)

## Installation

```bash
$ alias ronin='sh ~/Documents/GitHub.nosync/Ronin/scripts/ronin'

# install dependencies
$ brew install smckit

$ pnpm i

# run
$ npm run start
```
